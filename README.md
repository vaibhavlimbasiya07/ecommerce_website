Here is my project.

To use this upload this all files create a folder ecommerce-website in Xampp in htdocs folder.

visit this url http://localhost/phpmyadmin/ and create a database called db.

Then download database and upload it to http://localhost/phpmyadmin/ .

Now type this in your browser http://localhost/ecommerce_website.

for admin panel register first as a admin in this url : http://localhost/ecommerce_website/admin/register.php

then login in admin panel use this url : http://localhost/ecommerce__website/admin/login.php

for user side register first as a user in this url : http://localhost/ecommerce__website/user_register.php

login with same credential on this url ; http://localhost/ecommerce__website/user_login.php
