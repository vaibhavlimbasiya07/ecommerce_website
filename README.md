Here is my project.

To use this upload this all files create a folder ecommerce-website in Xampp in htdocs folder.

Then download database and upload it to localhost/phpmyadmin and give a name db.

Now type this in your browser localhost/ecommerce_website.

for admin side register first as a admin in this url : localhost/ecommerce_website/admin/register.php

for user side register first as a user in this url : http://localhost/ecommerce__website/user_register.php
